http와 https

Jan 17, 2019
1 minute read

Related Posts

http (Hyper Text Transfer Protocol)

  • Client와 Server가 메세지를 주고 받는 통신 규약
  • http://로 시작
  • 메세지를 암호화 하지 않음

https (Hyper Text Transfer Protocol Secure Socket Layer)

  • Client와 Server가 메세지를 주고 받는 통신 규약
  • https://로 시작
  • 메세지를 암호화함
  • 공개키, 개인키 암호화
  • SSL 인증서 구입 비용과 갱신 비용이 비싸다
  • 암호화로 속도가 느리다
