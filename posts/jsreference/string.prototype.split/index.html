String.prototype.split([separator[, limit]])

설명

split() 메서드 string 객체를 구분자(separator)을 이용해서 여러개의 문자열로 나눈 배열을 반환합니다.

매개변수

separatoroptional

원본 문자열을 나눌 구분자 문자열이나 정규표현식을 사용할 수 있다.

limitoptional

끊어진 문자열의 최대 개수

예제

빈 문자열인 경우

1
2
3

const str = '';
console.log(str.split());
// output: [""]

매개변수 사용법

1
2
3

const str = 'Hello World';
console.log(str.split(' ', 1));
// output: ["Hello"]
