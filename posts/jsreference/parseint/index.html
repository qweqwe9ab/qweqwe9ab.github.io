[JS Reference] parseInt()

Jul 06, 2021
1 minute read

parseInt(string, radix)

설명

매개변수

string는 문자열이 아니라면 문자열로 변환하고 선행 공백은 무시합니다.

radixoptional 2와 36 사이의 진수(default가 10이 아니다.)

string 이 “0x” 나 “0X"로 시작한다면, radix 는 16(16진) string 이 “0"으로 시작한다면, radix 는 8(8진)이거나, 10(십진)입니다. string 이 다른 값으로 시작 된다면, radix 는 10(십진)입니다.

반환값

주어진 string으로부터 분석한 radix진수의 정수를 반환합니다.

숫자로 시작하고, 중간에 숫자가 아닌 문자가 나오는 경우는 문자를 무시하고 앞까지의 숫자를 리턴한다.

NaN을 반환하는 경우: radix가 2보다 작거나 36보다 큰 경우 첫번째 non-whitespace 문자가 숫자로 변환되지 않는 경우
